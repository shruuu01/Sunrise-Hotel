# 🏨 Sunrise Hotel Management System

A web-based hotel management system designed to streamline room booking, billing, and customer management operations. The system provides an efficient and user-friendly interface for managing hotel services and customer records.

## ✨ Features

* 🛏️ Room Booking Management
* 👤 Customer Registration & Management
* 💳 Billing & Payment Processing
* 📋 Reservation Tracking
* 🔍 Customer Record Search
* 📊 Hotel Operations Management
* 📱 Responsive User Interface
* 🗄️ Secure Database Management

## 🛠️ Tech Stack

### 🎨 Frontend

* HTML5
* CSS3
* JavaScript

### ⚙️ Backend

* PHP

### 🗄️ Database

* MySQL

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/shruuu01/Sunrise-Hotel.git
```

2. Move the project folder to your XAMPP `htdocs` directory.

3. Import the MySQL database into phpMyAdmin.

4. Start Apache and MySQL using XAMPP.

5. Open the project in your browser:

```text
http://localhost/Sunrise-Hotel
```
